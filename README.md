# Elodin

Elodin is a platform for rapid design, testing, and simulation of
drones, satellites, and aerospace control systems. This repository
is a collection of core libraries:

- `libs/nox`
- `libs/nox-ecs`
- `libs/nox-ecs-macros`
- `libs/conduit`
- `libs/xla-rs`
