# Features

This is the first version of our sandbox prototype that showcases the real-time editing experience of Elodin.

- `Python Code Editor` with a `3D Sim Output` viewer
  ![editor-demo](/assets/v010-editor-demo.gif)

- `Double Pendulum` template - take a look at [our tutorial](/tutorial/double-pend) or [create a sandbox](https://app.elodin.systems/sandbox/new/double-pend) using this template
  ![double-pendulum](/assets/v010-double-pendulum.gif)

- `Three-Body Problem` template - you can [try it](https://app.elodin.systems/sandbox/new/three-body) right away or check our the video to see how to implement it from scratch
