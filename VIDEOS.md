# Videos

A collection of explainer videos covering Elodin tools, workflows, and internals. Add new entries under the most relevant section (create a new section if none fits).

Each entry should be a clickable thumbnail followed by a short description. For YouTube videos, use the standard thumbnail URL `https://img.youtube.com/vi/<VIDEO_ID>/hqdefault.jpg` (or `maxresdefault.jpg` if available):

```markdown
[![Title](https://img.youtube.com/vi/VIDEO_ID/hqdefault.jpg)](https://www.youtube.com/watch?v=VIDEO_ID)

**Title** — One or two sentence description. By Author.
```
